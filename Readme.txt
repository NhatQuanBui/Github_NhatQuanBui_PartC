# Readme.txt

To Build and Test the 'hello_NhatQuanBui' Application:

Prerequisites:
- Python 3.x
- Make (for Windows-based systems)

Instructions:
1. Clone this repository to your local machine.
2. Open a terminal and navigate to the repository directory.
3. Run the following command to build, test, and package the application:
For example: make run , make install, make build, make test, make clean
4. The application will be compiled, unit tests will run, and a deployable package will be created.
5. You can find the package in the 'build' directory.
6. Cleaning Up:
- To clean up generated files, run: make clean

For Windows-based systems, you can adapt the instructions to use a different build automation tool.
